{%= __("Some notes on Translations") %}:

  • {%= __("Translations context introduced are as follows. Please note the Order of Precedence.") %}
    • key:doctype:name:fieldname ( for example frappe._(key, context="TODO:123123:description") )
    • *:doctype:name:fieldname ( for example frappe._("*", context="TODO:123123:description") )
    • key:doctype:name
    • key:parenttype:parent
    • key:doctype:fieldname
    • key:doctype
    • key:parenttype
    • key
  • {%= __("Translations can be fetched using the following utility functions") %}
    • {%= __("For fetching translations based on order of precedence mentioned above use renovation_core.utils.translate.get_doc_field_translation ") %}
    • {%= __("For fetching translated doc based on order of precedence use renovation_core.utils.translate.get_doc_translations ") %}

{%= __("Quick help on adding a Translation") %}:

By Selecting a filter (doctype , docname and docfield ) you decide the translation context.

For example selecting doctype, docname and docfield the context will be doctype:name:fieldname. Please check the notes above for the different ways of formulating the context.
